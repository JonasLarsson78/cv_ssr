# Vue SSR app

En enkel demo-app med Vite, Vue 3, TypeScript och server-side rendering (SSR).

## Utvecklingsläge (SSR)

```bash
npm install
npm run dev:ssr
```

Öppna sedan http://localhost:4173.

## Bygga klient + server-bundle

```bash
npm run build
```

(En enkel Node-server för produktion kan läggas till vid behov.)
